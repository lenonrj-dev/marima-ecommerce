# Backend Marima

## Requisitos
- Node.js 20+
- MongoDB rodando localmente ou remoto

## Setup
1. Copie `.env.example` para `.env`.
2. Instale dependências:
   - `npm install`
3. Rode seed inicial:
   - `npm run seed`
4. Suba em desenvolvimento:
   - `npm run dev`

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run seed`

## API
Base URL: `http://localhost:4000/api/v1`
